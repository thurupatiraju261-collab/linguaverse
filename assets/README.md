# Assets Folder

This folder is reserved for additional media used by LinguaVerse beyond the
core brand assets in `/images` (logo & favicon):

- Course thumbnails, banner photography, or downloadable PDFs referenced by
  the "Resources" cards on `language-details.html`.
- Social share preview images (Open Graph / Twitter cards).
- Any future audio clips for pronunciation practice.

All current resource links in the app (e.g. "Beginner Course", "Grammar
Handbook") are intentionally placeholders (`href="#"`) — drop real files in
here and update the links in `js/languages-data.js` to go live.
